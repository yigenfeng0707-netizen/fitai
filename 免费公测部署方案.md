## 🚀 推荐方案：Oracle Cloud 永久免费

### 为什么选择Oracle Cloud？
- ✅ **永久免费**，不是试用
- ✅ **4核24GB ARM** 服务器（超强配置）
- ✅ **200GB存储空间**
- ✅ **10TB/月流量**
- ✅ 支持Docker部署

### Oracle Cloud 部署步骤

#### 第1步：注册Oracle Cloud

1. 访问 https://www.oracle.com/cloud/free
2. 点击"Start for free"
3. 填写信息（需要信用卡验证，不会扣费）
4. 选择区域（推荐：日本东京 或 韩国首尔，国内访问快）

#### 第2步：创建ARM实例

1. 登录Oracle Cloud控制台
2. 点击"计算" → "实例" → "创建实例"
3. 配置：
   - 名称：`fitai-server`
   - 映像：**Canonical Ubuntu 22.04** (选择ARM)
   - 配置：**VM.Standard.A1.Flex** (4 OCPU, 24GB内存)
   - SSH密钥：下载或使用已有
4. 点击"创建"

#### 第3步：配置安全列表

1. 进入实例详情页
2. 点击"虚拟云网络" → "安全列表"
3. 添加入站规则：
   - 端口 80 (HTTP)
   - 端口 443 (HTTPS)
   - 端口 8000 (API)

#### 第4步：连接服务器并部署

```bash
# SSH连接（使用下载的密钥）
ssh -i your-key.pem ubuntu@你的公网IP

# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker ubuntu

# 安装Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/download/v2.20.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

# 创建项目目录
mkdir -p ~/fitai && cd ~/fitai

# 克隆代码
git clone https://gitee.com/你的用户名/fitai.git .

# 配置环境变量
cp .env.example .env
nano .env  # 编辑配置

# 启动服务
docker-compose up -d --build

# 查看状态
docker-compose ps
```

#### 第5步：配置域名（可选）

如果你有域名，可以使用Cloudflare免费CDN：
1. 注册Cloudflare：https://dash.cloudflare.com/sign-up
2. 添加你的域名
3. 修改域名DNS为Cloudflare提供的NS
4. 添加A记录指向服务器IP
5. 开启"橙色云朵"（CDN代理）
6. 在SSL/TLS设置中选择"完全"

## 🎯 方案二：Railway 一键部署（最简单）

### Railway特点
- ✅ 无需服务器配置
- ✅ 自动HTTPS
- ✅ 支持Docker
- ✅ 每月$5免费额度（够用）

### 部署步骤

#### 1. 准备GitHub仓库

```bash
# 在本地
cd d:\fyglx
git add .
git commit -m "准备Railway部署"
git push origin master
```

#### 2. 注册Railway

1. 访问 https://railway.app
2. 使用GitHub账号登录
3. 点击"New Project"

#### 3. 部署后端

1. 选择"Deploy from GitHub repo"
2. 选择你的仓库
3. 配置环境变量（Settings → Variables）
4. 自动部署

#### 4. 添加数据库

1. 在项目中点击"+"
2. 选择"PostgreSQL"
3. 自动创建并连接

#### 5. 添加Redis

1. 再次点击"+"
2. 选择"Redis"
3. 自动创建并连接

#### 6. 配置域名

1. 进入服务Settings
2. 点击"Generate Domain"
3. 获得免费域名如：`fitai.up.railway.app`

## 🔧 快速部署脚本

### Oracle Cloud 一键部署

创建 `deploy-oracle.sh`：

```bash
#!/bin/bash

# Oracle Cloud FitAI 一键部署脚本

echo "🚀 开始部署 FitAI..."

# 1. 更新系统
sudo apt update && sudo apt upgrade -y

# 2. 安装Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 3. 安装Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/download/v2.20.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

# 4. 克隆代码
git clone https://gitee.com/你的用户名/fitai.git ~/fitai
cd ~/fitai

# 5. 配置环境变量
cat > .env << 'EOF'
APP_ENV=production
APP_SECRET_KEY=$(openssl rand -hex 32)
DB_HOST=localhost
DB_PORT=5432
DB_USER=fitai
DB_PASSWORD=$(openssl rand -hex 16)
DB_NAME=fitai_db
REDIS_HOST=localhost
REDIS_PORT=6379
EOF

# 6. 启动服务
docker-compose up -d --build

# 7. 检查状态
docker-compose ps

echo "✅ 部署完成！"
echo "访问: http://$(curl -s ifconfig.me)"
```
