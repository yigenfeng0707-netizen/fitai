# FitAI - 健身/瑜伽/教培商用级AI智能管理系统

🎉 **完整的商用级健身管理系统，支持AI智能助手！**

---

## 🌟 功能模块

### 核心功能
- 👥 **会员管理** - 会员注册、信息维护、会员卡管理
- 📚 **课程管理** - 课程创建、排课、课程表
- 📅 **预约管理** - 在线预约、签到签退、取消预约
- 👨‍🏫 **教练管理** - 教练信息、排班、绩效统计
- 💰 **财务管理** - 收支记录、报表统计、退款处理
- 📊 **数据分析** - 可视化图表、经营报表、数据导出

### AI智能功能
- 🤖 **AI智能助手** - DeepSeek大模型支持，智能客服
- 📝 **智能推荐** - 个性化课程和教练推荐
- 💬 **智能问答** - 常见问题自动解答

### 系统特性
- 🔐 **安全认证** - JWT双Token、RBAC权限模型
- 🛡️ **多租户隔离** - 数据完全隔离，互不干扰
- ⚡ **限流防护** - 防暴力破解、API限流
- 📝 **审计日志** - 完整操作记录
- 🔄 **容错机制** - 熔断器、自动重试
- 💾 **数据备份** - 自动备份与恢复

---

## 🛠️ 技术栈

### 后端
- **框架**: FastAPI + SQLAlchemy
- **数据库**: PostgreSQL
- **缓存**: Redis
- **认证**: JWT + RBAC
- **部署**: Docker

### 前端
- **框架**: React 18 + TypeScript
- **UI库**: Ant Design Pro
- **图表**: ECharts
- **状态管理**: Umi

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://gitee.com/fengyigen/fitai.git
cd fitai
```

### 2. 本地开发

```bash
# 后端
cd backend
pip install -r requirements.txt
python start_server.py

# 前端（已有standalone版本）
# 直接打开 frontend/dist/FitAI-Standalone.html
```

### 3. Docker部署

```bash
docker-compose up -d
```
