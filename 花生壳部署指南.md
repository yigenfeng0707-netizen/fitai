## 📝 部署步骤

### 第一步：注册花生壳账号

1. 访问 https://www.oray.com/peanut
2. 点击 **免费注册**
3. 使用手机号或邮箱注册账号
4. 登录后进入花生壳管理控制台

### 第二步：下载并安装花生壳客户端

#### Windows系统
1. 在花生壳官网下载Windows客户端
2. 安装并打开客户端
3. 使用刚才注册的账号登录

#### Linux系统（服务器部署）
```bash
# 下载花生壳客户端
wget https://download.oray.com/peanut/peanut_linux_x86_64.tar.gz

# 解压
tar -zxvf peanut_linux_x86_64.tar.gz

# 进入目录
cd peanut_shell

# 启动客户端
./phtunnel
```

### 第三步：启动本地服务

#### 方式1：直接启动前端（最简单，无需后端）

```bash
# 直接用浏览器打开前端Standalone版本
# 路径：d:\fyglx\frontend\dist\FitAI-Standalone.html

# 启动一个简单的HTTP服务器（可选）
# Python 3
cd d:\fyglx\frontend\dist
python -m http.server 8080

# Node.js (如果已安装)
cd d:\fyglx\frontend\dist
npx serve -p 8080
```

#### 方式2：完整后端部署（如果需要）

```bash
cd d:\fyglx\backend

# 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 启动后端
python start_server.py
# 后端默认运行在 http://127.0.0.1:8000
```

### 第四步：配置花生壳内网穿透

1. 打开花生壳客户端
2. 点击 **添加映射**
3. 配置映射：
   - **映射类型**：HTTP 或 HTTPS
   - **内网端口**：
     - 如果用前端Standalone：`8080`
     - 如果用完整后端：`8000`
   - **内网IP**：`127.0.0.1`
4. 点击 **添加**

### 第五步：获取公网访问地址

映射创建成功后，花生壳会提供一个免费域名，例如：
```
https://你的用户名.oicp.vip
```

## 🔧 常见问题

### Q1: 花生壳启动后无法访问？
**排查步骤**：
1. 检查本地服务是否正常运行
2. 检查防火墙是否阻止
3. 检查映射配置是否正确

### Q2: 速度慢怎么办？
- 免费版有速度限制，可考虑付费版
- 或者等待花生壳分配更快的节点

### Q3: 不想用花生壳，还有其他选择吗？

- **FRP自建**（需要有公网IP的服务器）
